Detailed Description

The contents in this class inherits the first name, last name, social insurance number, and date of birth from the Employee class (the parent) and will validate and get specific information intended for a Contract employee that will be inputed.
Exception Strategy: When validating the information for content towards the Contract Employee Class each property for the specific variable will validate the incoming value. Throwing an exception if an error occurs.
