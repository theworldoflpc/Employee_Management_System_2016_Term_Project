Detailed Description

Displays information to the screen to the user. This class acts as the View Layer.
Exception Strategy: When the user is prompted to enter any date, there may be an exception thrown when it is converted to a DateTime variable. If this happens, the exception is caught and the user is allowed to either a) Re-enter the date in the correct format b) Quit the process
