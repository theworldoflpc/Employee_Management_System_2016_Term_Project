Detailed Description

This is the parent class for all the other child employee classes. The contents in this class will be used to get the user's First name, Last name, Social Insurance Number, and their date of birth.
Exception Strategy: When validating the information for content towards the Parent Employee Class each property for the specific variable will validate the incoming value. Throwing an exception if employee fails to an error occurs.
